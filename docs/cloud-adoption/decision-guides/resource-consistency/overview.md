## <a name="deployment-consistency"></a><span data-ttu-id="1ff67-122">Consistência de implantação</span><span class="sxs-lookup"><span data-stu-id="1ff67-122">Deployment consistency</span></span>

## <a name="policy-consistency"></a><span data-ttu-id="1ff67-130">Consistência de política</span><span class="sxs-lookup"><span data-stu-id="1ff67-130">Policy consistency</span></span>

<span data-ttu-id="1ff67-131">Para garantir que as políticas de controle sejam aplicadas quando os recursos são criados, a parte do design de agrupamento de recursos envolve o uso de uma configuração comum ao implantar recursos.</span><span class="sxs-lookup"><span data-stu-id="1ff67-131">To ensure that governance policies are applied when resources are created, part of resource grouping design involves using a common configuration when deploying resources.</span></span>
