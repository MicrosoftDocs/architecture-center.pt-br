<span data-ttu-id="583f1-103">Com frequência, as soluções de Big Data usam trabalhos em lotes de execução longa para filtrar, agregar e, de outro modo, preparar os dados para análise.</span><span class="sxs-lookup"><span data-stu-id="583f1-103">Big data solutions often use long-running batch jobs to filter, aggregate, and otherwise prepare the data for analysis.</span></span> <span data-ttu-id="583f1-104">Normalmente, esses trabalhos envolvem a leitura de arquivos de origem de um armazenamento escalonável (como o HDFS, Azure Data Lake Store e Armazenamento do Azure), seu processamento e gravação da saída em novos arquivos no armazenamento escalonável.</span><span class="sxs-lookup"><span data-stu-id="583f1-104">Usually these jobs involve reading source files from scalable storage (like HDFS, Azure Data Lake Store, and Azure Storage), processing them, and writing the output to new files in scalable storage.</span></span>
